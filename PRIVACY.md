# CH Navigator — Privacy Policy

**Last updated:** 2026-05-05

CH Navigator is a Chrome extension that helps users open common Sitecore Content Hub pages quickly.

## What the extension accesses

- **The URL of the currently active tab**, only when you click the extension icon. The URL is used to detect the Content Hub tenant host (e.g. `acme.sitecorecontenthub.cloud`) so the extension can build links pointing at that same tenant.

## What the extension does NOT do

- It does not collect, store, transmit, or share any personal data.
- It does not read page contents, cookies, form fields, or authentication tokens.
- It does not contact any server controlled by the developer.
- It does not use analytics, tracking, or advertising SDKs.

## Data retention

Because nothing is collected, nothing is retained.

## Permissions

- `activeTab` — grants temporary access to the active tab's URL when you click the extension icon. Used solely to read the tenant origin (scheme + host).
